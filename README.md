# Bizness - Intelligent MSME OS

A modern, intelligent operating system for MSMEs (UMKM) combining POS, Inventory, AI Analytics, and Multi-business management.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack

- Next.js 14+ (App Router)
- TypeScript
- Tailwind CSS
- Zustand (State Management)
- Framer Motion (Animations)
- Recharts (Charts)
- Shadcn/UI Components
- Lucide React (Icons)
